# Requisitos No Funsionales
1.- Tiempo de respuesta: "La aplicación debe cargar la página principal en menos de 2 segundos bajo condiciones normales de red".
2.- Diseño Responsivo: "La interfaz debe adaptarse automáticamente a pantallas de dispositivos móviles, tablets y escritorio (Mobile First)".
# Requisitos Funcionales
1.- El sistema debe permitir el inicio de sesión y validar las credenciales contra la base de datos.
2.- El sistema debe permitir al usuario crear nuevos registros (ej: publicaciones, productos, tareas).
3.- El sistema debe proporcionar una barra de búsqueda para localizar elementos por nombre o categoría.
4.- El sistema debe permitir la eliminación lógica o física de registros.
